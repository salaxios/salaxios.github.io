Pleased to announce the creation and use of the Salaxios Narrative Engine Toolkit.

Including a native Javascript Editor titled JavaScript Game Scripting System - in Engine.

![jsgss](/assets/images/4/jsgss.png)

A Javascript Virtual Machine for storing code libraries and dialog - The Salaxios Game Text Database System (SGTDBS).

![jsgss](/assets/images/4/sgtdbs.png)

And a Parallax editor & collision editor for the Open Source RPG Maker MZ Altimit Movement Pixel Movement Engine - COLLED.

![COLLED](/assets/images/4/colled.png)

All 3 tools are part of a proprietary in-house patch to RPG Maker MZ titled the RPG Maker MZ + Salaxios Narrative Engine. The patch may be available for purchase when `the Locust` is finished.
